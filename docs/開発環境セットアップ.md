# 開発環境セットアップ

## .env

`.env.example` をコピーして `DISCORD_TOKEN` を設定する．

```bash
cp .env.example .env
```

## 起動

Docker Compose で DB・マイグレーション・ボットをまとめて起動する．

```bash
docker compose up -d --build
```

マイグレーションは `migrate` サービスが自動で適用する．

## ログ確認

```bash
docker compose logs -f bot
```

## 停止

```bash
docker compose down
```
