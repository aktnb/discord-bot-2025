# 開発環境セットアップ

## DB

Docker を使用する．`docker-compose.yml` を起動し、migration file を適用すれば良い．

#### Docker Compose の起動

```bash
docker compose up -d
```

#### Migration File の適用

```bash
migrate -path db/migrations -database "postgres://bot:botpass@localhost:5432/botdb?sslmode=disable" up
```

## .env

`.env.example` をコピーして DISCORD_TOKEN を設定する．

```bash
cp .env.example .env
```

## 起動

```bash
go run cmd/bot/main.go
```
